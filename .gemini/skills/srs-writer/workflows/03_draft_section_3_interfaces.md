# Workflow: Bước 3 - Soạn thảo Phần 3 (Yêu cầu Giao diện & Tích hợp)

Sau khi chốt phần tổng quan, tiếp tục viết phần 3: External Interface Requirements.

- **User Interfaces:** Mô tả các đặc điểm giao diện người dùng (chuẩn UI/UX, responsive, liệt kê các màn hình chính/Dashboard cần có).
- **Hardware Interfaces:** Mô tả tương tác phần cứng nếu có (máy in, máy quét mã vạch, thiết bị đọc thẻ CCCD...).
- **Software Interfaces:** Liệt kê các kết nối với phần mềm bên thứ 3 (Ví dụ: Database, API Hóa đơn điện tử, Zalo OA...).
- **Communications Interfaces:** Các giao thức truyền thông cần thiết (HTTPS, Webhooks, v.v.).

> **Lưu ý:** Thay thế các placeholder `<...>` bằng nội dung thực tế. Nếu thiếu thông tin, tự động đưa ra giả định (Assumption) và đánh dấu `[Cần xác nhận]`.
